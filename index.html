Mini curso de HTML

HyperText Markup Language (HTML) essa linguagem usada para organizar nosso arquivo de texto. Essa linguagem ajuda nossos navegadores a entender e exibir os arquivos do site.
Tudo em nosso site, e todos os outros sites, existem dentro de elementos HTML.
Os elementos são pedaços dessa linguagem que têm um significado e ajudam a descrever tudo o que colocamos na página.
